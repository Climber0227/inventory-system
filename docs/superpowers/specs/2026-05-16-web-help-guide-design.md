# 系统使用说明页面设计

## 概述

在 Web 端新增一个「使用说明」菜单页面，为管理员和员工提供完整的系统操作指南，涵盖所有模块的功能说明、操作流程、字段释义和注意事项。

## 页面位置

- 独立的一级菜单项，标题为「使用说明」，图标为 `Document` 或 `Reading`
- 放在侧边栏菜单中，所有角色（管理员和员工）均可访问
- 路由路径：`/help`

## 页面布局

采用「左侧目录树 + 右侧内容区」的两栏布局：

- **左侧目录树**（固定宽度 220px）：可折叠/展开的树形结构，点击节点滚动到对应内容区
- **右侧内容区**（自适应）：卡片式内容展示，每个章节为一个可折叠面板
- **顶部搜索框**：全文搜索关键词，匹配到的内容高亮显示

### 页面状态

| 状态 | 表现 |
|------|------|
| 加载中 | Skeleton 骨架屏 |
| 空内容 | 不适用（内置静态内容） |
| 搜索无结果 | 显示 "未找到相关内容" + 建议关键词 |
| 搜索有结果 | 滚动到第一个匹配位置并高亮 |

## 内容结构

共 14 个章节，每个章节包含：功能概述、操作流程、字段说明、状态流转、注意事项。

### 一、工作台
- 统计卡片：商品总数、仓库数量、今日入库/出库
- 图表：近 30 天每日入库/出库趋势、本月累计曲线
- 快捷操作入口
- 最近动态列表
- 库存预警

### 二、商品管理
- 商品列表：字段释义（编码/名称/规格/单位/采购价/销售价/安全库存）、操作按钮、筛选条件
- 条码查看与打印
- 商品分类：树形结构、增删改查
- 商品状态（启用/停用）

### 三、供应商管理
- 供应商列表、新建/编辑表单字段说明
- 联系人信息管理

### 四、客户管理
- 客户列表、新建/编辑表单字段说明

### 五、仓库管理
- 多级仓库结构（1~4 级）
- 仓库树的概念、层级含义
- 新建/编辑仓库时的父子级选择

### 六、采购入库
- **列表页**：字段释义、筛选条件、操作按钮说明
- **新建/编辑**：表单字段、商品明细添加方式、草稿 vs 提交的区别
- **详情页**：查看完整信息
- **状态流转图**：草稿(0) → 待审批(4) → 已入库(1)；草稿(0)/待审批(4) → 驳回→草稿(0)；草稿(0)/已入库(1) → 取消；草稿(0)/已取消(2) → 作废
- **审核流程**：管理员审核通过/驳回，驳回需填写原因
- **导出**：全部导出 / 选中导出
- **批量作废**（仅管理员）

### 七、销售出库
- 与采购入库结构一致，增加客户/销售员特有字段说明
- 注意事项：审核通过后将扣减对应仓库库存

### 八、库存管理
- **库存列表**：按仓库树展示、展开/折叠、按仓库筛选
- **库存流水**：按时间倒序、搜索（商品/单号）、操作类型（入库/出库/调整）
- **库存预警**：低于安全库存的商品高亮显示

### 九、库存盘点
- 盘点方式：全盘 vs 抽盘
- 盘点流程：新建盘点单 → 录入实盘数量 → 审核通过 → 执行调整
- 差异数概念：系统库存与实盘数量的差异
- 操作限制：调整操作仅管理员可执行

### 十、库存调拨
- 调拨流程：选择调出/调入仓库 → 添加商品 → 提交 → 审核 → 完成
- 库存校验：调出时校验可用库存是否充足
- 状态流转：草稿(0) → 待审批(4) → 已完成(1)；草稿(0) → 取消；已取消(2) → 作废

### 十一、统计报表
- 入库汇总：30 天柱状图 + 汇总数据
- 出库汇总：同上
- 库存周转率：各商品周转率计算
- 库存预警列表

### 十二、系统管理（仅管理员）
- **员工管理**：新建/编辑员工、权限设置（管理员 vs 员工）、改密、启用/禁用
- **操作日志**：所有用户的操作记录、按模块/时间筛选
- **系统配置**：关键配置项管理
- **已作废列表**：所有已作废单据的统一查看入口

### 十三、权限说明
- 管理员角色（role_1）：全部权限，包括审核、作废、员工管理、系统配置
- 员工角色（role_2）：基础操作权限（新建、编辑草稿、查看、导出）
- 权限对照表：分模块列出管理员/员工的差异

### 十四、常见问题
- 单据审核流程是怎样的？
- 审核通过后库存什么时候更新？
- 为什么某些按钮不可点击？
- 数据删除和作废有什么区别？
- 多级仓库如何理解？

## 技术实现

### 路由配置
```ts
{
  path: '/help',
  name: 'Help',
  component: () => import('../views/help/HelpGuide.vue'),
  meta: { title: '使用说明' },
}
```

### 菜单配置
```ts
{
  path: '/help',
  title: '使用说明',
  icon: 'Reading',
}
```

### 文件结构
```
src/views/help/
  HelpGuide.vue          # 主页面（左侧目录 + 右侧内容）
  data/guideData.ts      # 所有章节内容的配置数据（JSON 格式）
```

### 数据方案
- 所有内容作为静态 JSON 数据存放在 `guideData.ts` 中
- 章节采用树形结构，支持无限层级嵌套
- 每章节有唯一 `id` 用于目录跳转锚点
- 搜索在前端对静态数据进行全文匹配

### 内容数据结构
```ts
interface GuideSection {
  id: string
  title: string
  icon?: string
  children?: GuideSection[]
  content: GuideContent[]
}

interface GuideContent {
  type: 'text' | 'table' | 'flow' | 'note' | 'warning' | 'image'
  title?: string
  body: string | TableData | FlowData
}
```

### 交互细节
- 左侧目录跟随滚动高亮当前章节
- 点击目录项平滑滚动到对应位置
- 右侧每个章节可折叠/展开，默认全部展开
- 搜索时折叠所有非匹配章节，展开匹配章节并高亮关键词
- 表格类内容：字段名 + 释义 + 示例
- 状态流转：使用文字描述流程图（带箭头符号）

## 非功能需求

- 所有内容静态加载，无网络请求
- 搜索响应时间 < 100ms
- 页面首次加载 < 1s（无外部依赖）
- 支持侧边栏折叠时自适应
